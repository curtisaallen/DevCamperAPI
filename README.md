# DevCamperAPI
DevCamper is a Node.js API that uses MongoDB and Express

![](https://curtisaallen.github.io/HooksNews/bg.png)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
